# Novel_short_primer_16S

This repository contains the analysis workflow and figure-generation scripts for the novel short primer manuscript.

The manuscript PDF included in this repository is `Novel short primers for next-generation.pdf`.

## Repository purpose

The repository rebuilds metadata, matrix files, taxonomic annotation tables, QIIME helper files, and manuscript figures from the retained processed inputs in `data_tables/` and `sh/`.

## Data

The active repository uses:
- metadata and derived matrix files in `data_tables/`
- retained original QIIME output tables in `data_tables/original/`
- helper shell and manifest files in `sh/`
- raw sequence data (FASTQ) files were submitted to the Sequence Read Archive (SRA) of the National Center for Biotechnology Information (NCBI) under BioProject `PRJNA1198036`

## File descriptions

- `0_2_FastQ_2_meta_mani_tsv.R`
  Creates `manifest.tsv` and `metadata.tsv` for the QIIME import workflow from `data_tables/meta.csv`.

- `0_Pear_Summary_vplE2.R`
  Summarizes PEAR merge results and creates file-name, repooling, and merge-summary tables from `data_tables/Pear_Summary.txt`.

- `1_1_Table_Phylum_to_meta.R`
  Builds `data_tables/meta.csv` from the sample IDs present in `Table_Phylum.tsv`, including a merged `file_name` and `QIIME_reads`.

- `2_Feature_Tables_to_Rarefied.R`
  Creates raw, rarefied, and filtered matrices from the retained QIIME feature tables.

- `3_ASV_Taxa.R`
  Creates `ASV_taxa.csv` from `taxa_data.tsv` and `sequences.fasta`, and summarizes sequence-length distributions.

- `Figure_1B_melting_temp.R`
  Creates the melting-temperature figure.

- `Figure_2_5_alpha_diversity_stats.R`
  Creates the alpha-diversity and related summary figure outputs.

- `Figure_3_Correlation.R`
  Creates the Standard-versus-Truncated correlation figure.

- `Figure_4A_4B_6A_6B_abundance.R`
  Creates the abundance figure panels.

- `Figure_4C_6C_ordination.R`
  Creates the ordination figure panels and their PERMANOVA and PERMDISP annotations.

- `data_tables/raw_matrix/`
  Contains raw matrix files generated from the retained QIIME tables.

- `data_tables/rarefied_matrix/`
  Contains rarefied matrices generated by the run-first matrix script.

- `data_tables/filtered_matrix/`
  Contains filtered matrices used as the active downstream inputs listed in `matrix_names.csv`.

- `data_tables/original/`
  Contains the retained processed QIIME exports and supporting classification files used to rebuild the repository outputs.

- `sh/`
  Contains saved manifest, metadata, and shell workflow helpers used for the QIIME processing path.

## Running from scratch

1. Run `1_1_Table_Phylum_to_meta.R`
2. Run `0_2_FastQ_2_meta_mani_tsv.R` if you need to rebuild the QIIME manifest and metadata files
3. Run `2_Feature_Tables_to_Rarefied.R`
4. Run `3_ASV_Taxa.R`
5. Run the figure scripts as needed

The generated matrix outputs are already present in the repository, so rerunning the preprocessing scripts is only necessary if you want to rebuild them.
